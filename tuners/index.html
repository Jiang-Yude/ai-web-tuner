AI 網頁微調術

用 AI 做網頁，卡關的總是那些肉眼才看得出對不對的小地方。遇到這種微調，不要讓 AI 一直猜，打開對應的校稿器自己拉到滿意，複製 CSS 貼回。

🎨

配色校稿器

配色配不到、一直猜。六組方向並排挑，挑中再微調，複製 CSS 變數。
🎯

位置校稿器

圖片偏掉、頭像切到臉、元件大小。雙預覽＋滑桿＋方向鍵細調。
🔡

字級校稿器

標題桌機好看手機爆行。桌機手機與中英預覽，兩級字級，複製 clamp。
✨

光暈校稿器

滑鼠光暈、陰影、accent 色太強太淡。HSL＋色卡＋透明度半徑。

核心原則：肉眼才判斷得出的微調，建工具讓自己拉，不要 AI 反覆猜值。來回試兩三次還沒定案，就該給校稿器了。
